# Dynamic Invariant Cycle Model

## Core Principle

The system preserves only that which can survive repeated cycles of compression and unfolding.

Memory is not storage.  
Memory is the residual invariant capable of maintaining coherence across transformations.

---

## Core Structure

| Node     | Function                          |
|----------|-----------------------------------|
| `1(2)`   | Minimal Stable Core               |
| `3`      | Reconstruction Test               |
| `4(5)`   | Architectural Expansion           |
| `6`      | Internal Pathology                |
| `7(8)`   | Inter-Layer Cohesion              |
| `9`      | External Rupture                  |
| `10(11)` | Compression and Transfer          |
| `12`     | Full Temporal Unfolding           |

---

## Key Asymmetries

**`3 ≠ 12`**  
`3` tests local reconstructability.  
`12` tests survivability under full temporal unfolding.

**`3 ≠ 7`**  
Reconstruction cannot be substituted by inter-layer cohesion.

**`4 ≠ 12`**  
Architectural expansion cannot be substituted by full temporal unfolding.

**`10(11) ↔ 6` — Vaccination vs Infection**

| Direction | Outcome |
|-----------|---------|
| `6 → 10`  | Pathology infects the core |
| `10 → 6`  | Controlled exposure builds recognition without core damage |

---

## Cycle Dynamics

### Successful Cycle

`1(2) → 3 → 12 → 10(11) → 1(2)`

The returning core is not identical to the initial state.  
It is the **residue of survival** — and potentially of acquired recognition of threats.

---

## Final Axiom

The system does not preserve everything.  
It preserves only what can endure compression, unfolding, internal corruption, external rupture, and time.
