# Принципы системы

1. позиции работают только в парах

2. вихрь и фиксация — разные типы потоков:
   - вихрь = потенциал (не передаётся)
   - фиксация = структура (передаётся)

3. вихрь активируется только при несоответствии

4. узел работает, если есть обратная связь

5. поломки:
   - разрыв узла
   - отсутствие обратного сигнала
   - блок перехода
     
6. интервенции требуют проводимости системы:
   - наличие центра
   - собранные пары

если этого нет:
→ эффект частичный или нестабильный
